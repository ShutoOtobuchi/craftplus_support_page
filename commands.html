よく使うコマンド一覧

プレイヤーをテレポートさせたいとき

コマンド 使用状況・用途
/tp @s @c プレイヤーをエージェントの元にテレポートさせる
/tp @s 0 0 0 プレイヤーをワールド座標0 0 0にテレポートさせる
※注意：
このコマンドは、/tp @目的地 @テレポートさせる対象という形になっているので、状況に応じてアレンジしてください。

エージェント（ロボット）をテレポートさせたいとき

コマンド 使用状況・用途
/tp @c @s エージェントをプレイヤーの元にテレポートさせる
/tp @c 0 0 0 エージェントをワールド座標0 0 0にテレポートさせる

ゲームモードをクリエイティブに変更したいとき

コマンド 使用状況・用途
/tag @s remove classmode プレイヤーについているclassmodeタグを取り外す
/gamemode c ゲームモードをクリエイティブに変更する
※注意：
クラフトプラスではすべての教材で、正誤判定やゲームモード変更の制限など授業を円滑に進めるため、プレイヤーにclassmodeタグというものを付与しています。ゲームモードを変更する際はclassmodeタグを取り外してから行ってください。

ゲームモードをアドベンチャーに戻したいとき

コマンド 使用状況・用途
/gamemode d ゲームモードをアドベンチャーに戻す
/tag @s add classmode プレイヤーにclassmodeタグを与える

ワールドにバグが発生したとき

コマンド 使用状況・用途
/function stage1/clear ステージ１を強制的にクリアする
/function stage1/clear_substage ステージ１のサブステージを強制的にクリアする
/function stage1/init ステージ１の状態を初期化する
/function stage1/reset サブステージの状態を初期位置に戻す
/function stage1/start ステージ１を強制的にスタートさせる
/function stage1/start_substage ステージ１のサブステージを強制的にスタートさせる

※ 各コマンドの stage の後ろの数字は、問題が起きたステージ番号に適宜変更してください。
